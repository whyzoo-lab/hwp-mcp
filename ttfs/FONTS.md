# 폰트 메트릭스 데이터베이스 가이드

## 왜 폰트 메트릭스 DB가 필요한가?

HWP 문서의 정확한 렌더링을 위해서는 **폰트 메트릭스**(글자 폭, 높이, 자간, 베이스라인 등) 데이터가 필수적이다.

HWP 문서는 특정 폰트를 지정하여 작성되며, 한컴 오피스는 해당 폰트의 메트릭스를 기준으로 줄 나눔, 페이지 분할, 표 셀 크기를 계산한다. 동일한 메트릭스를 사용하지 않으면 줄 바꿈 위치가 달라지고, 페이지 수가 변하며, 표 레이아웃이 무너진다.

```
원본 HWP 문서 (한컴바탕, 10pt)
  ↓
폰트 메트릭스 DB에서 "한컴바탕 10pt" 의 글자 폭 조회
  ↓
줄 나눔, 페이지 분할 계산
  ↓
웹 배포 시 → 폴백 폰트(Pretendard 등)로 대체 렌더링
```

즉, **메트릭스 DB는 레이아웃 계산에 사용**되고, **실제 렌더링은 폴백 폰트로 대체**된다. 저작권 폰트 파일을 사용자에게 배포할 필요가 없다.

## 저작권을 위배하지 않는 구축 방법

### 원칙

1. **폰트 파일(.ttf/.woff2)은 Git에 포함하지 않는다** — 저작권 보호 대상
2. **폰트 메트릭스 수치(숫자 데이터)는 저작권 보호 대상이 아니다** — 사실의 나열은 저작물이 아님
3. **메트릭스 DB만 추출하여 코드에 포함한다** — 합법

### 구축 절차

#### 1단계: 로컬에 폰트 설치

개발자 본인의 PC에 한컴 오피스를 정상 설치하면 폰트가 시스템에 설치된다.

- **Windows**: `C:\Windows\Fonts\` 또는 한컴 설치 경로
- **macOS**: `/Library/Fonts/` 또는 한컴 설치 경로
- **Linux**: 한컴 오피스 Linux 버전 설치 또는 별도 구매

> 한컴 오피스 체험판 또는 정식 라이선스로 설치한 폰트를 로컬 개발에 사용하는 것은 라이선스 범위 내이다.

#### 2단계: 메트릭스 추출

`ttfs/` 폴더에 폰트 파일을 배치한 후, rhwp의 네이티브 빌드에서 메트릭스를 추출한다.

```bash
# 네이티브 빌드 (폰트 메트릭스 DB 자동 구축)
cargo build

# SVG 내보내기로 메트릭스 정확도 검증
cargo run -- export-svg samples/your-test.hwp
```

rhwp는 빌드 시 `ttfs/` 폴더의 폰트를 읽어 글자 폭, 높이, 자간 등의 수치 데이터를 메모리에 로드한다.

#### 3단계: 웹 배포 시 폴백 처리

WASM 웹 빌드에서는 저작권 폰트를 포함하지 않고, 대응되는 오픈 라이선스 폰트로 폴백한다.

| 원본 폰트 | 폴백 폰트 | 라이선스 |
|----------|----------|---------|
| 한컴바탕, HY명조 | 시스템 세리프 또는 Noto Serif KR | OFL |
| 한컴돋움, HY고딕 | Pretendard 또는 Noto Sans KR | OFL |
| 함초롬바탕 | 시스템 세리프 | — |
| 함초롬돋움 | Pretendard | OFL |
| 맑은 고딕 | Pretendard | OFL |
| Arial, Calibri | 시스템 산세리프 | — |
| Times New Roman | 시스템 세리프 | — |

## 폰트 파일 목록

저작권 보호를 위해 Git에 포함하지 않으며, 개발자가 로컬에 별도로 설치해야 한다.

### ttfs/ (루트)

| 파일명 | 폰트명 | 용도 |
|--------|--------|------|
| hamchob-r.ttf | 함초롬바탕 Regular | HWP 기본 바탕체 메트릭스 |
| hamchod-r.ttf | 함초롬돋움 Regular | HWP 기본 돋움체 메트릭스 |

### ttfs/hwp/

한컴 오피스에 포함된 전용 폰트.

| 파일명 | 비고 |
|--------|------|
| HBATANG.TTF | 한컴바탕 |
| HDOTUM.TTF | 한컴돋움 |
| H2HDRM.TTF | HY헤드라인M |
| HY*.TTF | HY 시리즈 (명조, 신명조, 울릉도 등) |
| HAN*.ttf | 한양 시리즈 (백제, 솔, 소마 등) |
| ahn_*.ttf | 안상수체 |
| NanumBrush.ttf | 나눔손글씨 붓 |
| FZSong_Super.TTF | 중국어 송체 |

### ttfs/windows/

Windows 시스템 폰트. 운영체제에서 직접 참조한다.

## 요약

| 항목 | Git 포함 | 저작권 |
|------|---------|--------|
| 폰트 파일 (.ttf/.woff2) | **포함하지 않음** | 보호 대상 |
| 메트릭스 수치 (글자 폭, 높이) | 코드에 포함 가능 | 사실 데이터 (비보호) |
| 폴백 매핑 테이블 | 코드에 포함 | 자체 작성 |
| 오픈 라이선스 폰트 (Pretendard 등) | 포함 가능 | OFL/무료 |
